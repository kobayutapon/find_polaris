# find_polaris
天体望遠鏡の極軸合わせサポートツールです。北極星の大まかな位置を見つけます。PCに接続するとGPSとして利用できます。

# ハードウェア
- M5Stack Basic v2.7(https://www.switch-science.com/products/9010)
- M5 GNSS Module(https://www.switch-science.com/products/9276)

# ビルド環境
- Arduino IDE v2.3.6
- M5Stack v2.1.4
- M5Unified 0.2.10
- TinyGPS++ 0.0.4
- Madgiwick 1.2.0
